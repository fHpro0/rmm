== Description ==

Take full control of your website's remote metadata with our powerful WordPress plugin. 
Easily modify or remove remote metadata attached to your content for enhanced privacy, improved SEO, and optimized performance. 
Our intuitive interface empowers you to manage metadata efficiently, ensuring your website meets your exact specifications. 
Whether you're fine-tuning SEO parameters or enhancing data privacy, our plugin provides the flexibility and control you need. 
Streamline your content management process and boost your website's performance with our remote metadata management solution today!

== Installation ==

1. Install Plugin
2. Activate Plugin
3. Load a normal Site on your Wordpress System (Frontend User Site)
4. Go to Settings > Remote Metadata Modifier
5. Select the URLs that you want to either save to your wordpress system and load it from there or just remove it from the Head in the parsed HTML Output.
6. Create a Cronjob to automatically updated the content if there is new or just reload it manually.
